Detailed Description

This class provides a unified interface for processing all the real-time information provided by the SSL and our system on the physical state and the referee's current state of the game. Note this interface is NOT responsible for networking-related issues, e.g. all objects passed to it are assumed to be from a trusted source. Some packets may however still be discarded if they have bad timestamps for example.
