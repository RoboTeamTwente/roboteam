Detailed Description

template<typename T, size_t Cap>
class rtt::collections::static_vector< T, Cap >

Container that behaves like std::vector but is allocated on the stack with a max capacity.

Template Parameters
TType to be contained
Capsize_t, maximum capacity of the static_vector.

Opposing to std::vector iterators are never invalidated unless the entire structure is dropped.
