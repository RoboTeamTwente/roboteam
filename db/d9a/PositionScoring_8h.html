struct rtt::ai::stp::gen::PositionScores

Struct that is used to store computations made with this module. Save computations here that are usable for each robot for a position. DO NOT save values specific to a robot in here (like TimeToPosition). If a position's score for a specific evaluation already had been computed in the tick, it will use that value instead of recomputing it.
