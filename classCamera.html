◆ isPositionVisible()

bool Camera::isPositionVisible ( const Eigen::Vector3d &  fieldPoint,
double  marginFactor = 0.0 
) const

!! Position is in millimeters, NOT meters!! Checks if the position is visible. The marginFactor is subtracted from the image boundaries in pixels on each side And then it is checked if the coordinate produced falls within the acceptable range. E.g. if marginFactor is 0.1 and the image is 1280x1024 128 pixels are substracted from each side and only the inner region is considered visible. This function returns true if the image width and height are unknown
