Returns a string of 0bxxxxxxxx where x=1 if x in range [start, stop), else 0.

If reverse=True, swap 0 and 1 bitString(1, 2, reverse=False) = 0b01000000
bitString(5, 6, reverse=True ) = 0b11111011
