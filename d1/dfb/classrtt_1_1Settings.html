Detailed Description

Game settings that must be shared between all players and the referee (e.g. who is playing on which side, which team is yellow, etc.) These settings are set by the primary AI, and are read-only for secondary AIs. Importantly referee can also change these settings, thus this class must be thread-safe, since referee is handled in a separate (IO) thread.

The AI has to be able to handle any changes to these settings WITHOUT restart. Thus settings such as visions_port that require a restart of the AI, are not included here.
