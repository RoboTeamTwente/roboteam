Detailed Description

template<typename Tp, std::size_t Size>
class rtt::collections::circular_buffer< Tp, Size >

A standard container for Circular buffers. Note this class does not have iterators because they do not comply with the standard as there would be cases where first == last. This quickly becomes very ugly. The front of the buffer are the oldest elements, e.g. the ones first pushed in. These are the next to be erased.
