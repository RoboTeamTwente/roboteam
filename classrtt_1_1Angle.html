Detailed Description

The Angle class is a modular class on the interval [-PI, PI) that deals with addition, subtractions, distances and comparisons on this scale.

◆ rotateDirection()

bool rtt::Angle::rotateDirection ( const Angle &  other) const
noexcept

Check what is the shortest direction to move from this angle to the other angle. In case the distance between both angles is 0 then we prefer the positive direction, so in this case we return true. In case the distance between both angles is PI then we prefer the negative direction, so in this case we return false.

Parameters
otherThe other angle
Returns
True if the positive direction is the shortest (which is counterclockwise), false if the negative direction is the shortest (which is clockwise).
